# My Academia Buddy

## Overview

My Academia Buddy is an AI-assisted academic planning web application developed as part of the SEG2105 Software Engineering course at the University of Ottawa.

The application helps students organize their academic activities by managing courses, assignments, exams and automatically generating personalized study plans.

## Features

- Dashboard
- Course Management
- Assignment Management
- Exam Management
- Smart Study Planner
- Automatic Workload Estimation
- Automatic Course Difficulty Estimation
- Study Insights
- Study Progress Tracking
- Local Data Persistence

## Technologies

- React
- Vite
- JavaScript
- CSS
- Local Storage
